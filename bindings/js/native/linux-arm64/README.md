# @kurrent/gaffer-runtime-linux-arm64

Platform-specific shared library for [`@kurrent/gaffer-runtime`](https://www.npmjs.com/package/@kurrent/gaffer-runtime).

This package is pulled in automatically as an optional dependency when you install `@kurrent/gaffer-runtime` on a matching platform. Don't install it directly.
